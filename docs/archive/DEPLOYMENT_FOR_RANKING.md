## üöÄ DEPLOY TODAY - 3 Step Process

### Step 1: Deploy Contract (30 minutes)

**Prerequisites**:
- Private key ready
- Celo account with CELO for gas
- Forge installed

**Command**:
```bash
export PRIVATE_KEY=your_private_key_here
export CELO_RPC_URL=https://forno.celo.org

forge script contracts/script/Deploy.s.sol:DeployProcurement \
  --rpc-url $CELO_RPC_URL \
  --private-key $PRIVATE_KEY \
  --broadcast
```

**Expected Output**:
```
ProcurementV2 contract deployed at: 0x...
Token address: 0x765dE816845861e75a25FCA122BB6Caa78443cB5
```

**Save this address!** You'll need it for verification.

### Step 2: Verify Contract (15 minutes)

1. Go to https://explorer.celo.org
2. Search for your contract address
3. Click "Verify & Publish"
4. Upload source code from `contracts/src/core/ProcurementV2.sol`
5. Select Solidity version: 0.8.20
6. Compiler: Solc 0.8.28
7. Submit

**Why**: Shows you're a legitimate builder

### Step 3: Complete Self.xyz (20 minutes)

1. Go to https://self.xyz
2. Click "Get Verified"
3. Complete identity verification
4. Link your wallet
5. Confirm verification

**Why**: Unlocks "Self Verified" booster
